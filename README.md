---

## ✨ Core Architecture

### 🧠 Neural Itinerary Engine
- **Context-Aware Planning**: Generates comprehensive daily schedules based on your destination and timing.
- **Dynamic Adjustments**: Adapts to user preferences including travel style, pace, and dietary needs.
- **Smart Estimation**: Provides realistic cost and duration metrics for every activity.

### 🛡️ Elite Security Vault
- **Biometric-Grade Auth**: 50/50 split-screen authentication portal with atmospheric narrative elements.
- **Secure Sync**: Real-time synchronization across devices using Firebase's distributed architecture.
- **Privacy First**: Advanced data encryption and personalized security gates.

### 💎 Trillion-Dollar UI/UX
- **Glassmorphism Design**: High-contrast, transparent cards and ultra-modern typography.
- **Micro-Animations**: Fluid transitions powered by Framer Motion for a premium tactile feel.
- **Command Dashboard**: A central hub for managing your journeys, stats, and global explorations.

## 🛠️ Installation & Launch

### 1. Synchronize the Repository
```bash
git clone https://github.com/your-username/PARUL-HACKATHON.git
cd PARUL-HACKATHON
```

### 2. Install Neural Dependencies
```bash
npm install
```

### 3. Configure the Environment
Create a `.env.local` file in the root directory and populate it with your security keys:
```env
NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_auth_domain
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_storage_bucket
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id
GROQ_API_KEY=your_groq_api_key
```

### 4. Launch the Odyssey
```bash
npm run dev
```
Navigate to [http://localhost:3000](http://localhost:3000) to begin your journey.
